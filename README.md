# 구름 프론트엔드 깃허브 기초 실습
## 구성원: 장수민, 진서현, 정원준
### 실습 과제
- repo에 각자 아무 작업물이나 push해서 올리고, 다른 팀원들이 올린 내용을 pull 해보기.   
- 각자 작업용 branch를 생성해서 거기에 작업물들을 push 한 후에 동료들과 PR을 수행
- merge conflict 사례를 만들고 해결
